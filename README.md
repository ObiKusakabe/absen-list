# Cara Clone (Copy) project ini 🍎

1. Install git (Control Version System).
2. Memiliki Shell (CLI). Untuk Windows OS, bisa pakai Windows PowerShell.
3. Check klo git udah ke install dengan baik. Pakek command : "git version".
